📋 QUICK SETUP (3 steps):

1. Get FREE Gemini API Key:
   👉 https://aistudio.google.com/app/apikey
   
2. Update .env file:
   
   EXPO_PUBLIC_LLM_PROVIDER=gemini
   EXPO_PUBLIC_GEMINI_KEY=AIza...your_key_here
   EXPO_PUBLIC_GEMINI_MODEL=gemini-1.5-flash

3. Restart app:
   
   npx expo start -c

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 DEPLOY WITH GEMINI:

When deploying to Supabase, the setup script will ask which
provider you want. Choose option 1 (Gemini).

cd backend
./SETUP.sh

Or manually:

supabase secrets set \
  GEMINI_API_KEY=your_key \
  LLM_PROVIDER=gemini \
  GEMINI_MODEL=gemini-1.5-flash
